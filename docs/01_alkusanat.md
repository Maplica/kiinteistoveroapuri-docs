# Alkusanat

# KiinteistöveroApuri

KiinteistöveroApuri on väline kunnan kiinteistöveroselvityksen selkeyttämiseen. KiinteistöveroApuri muotoilee kiinteistötietojen sekä nykyisten kiinteistöverotietojen vertailun ja läpikäynnin selkeäksi kokonaisuudeksi ja auttaa tekemään työstä tehokasta.

KiinteistöveroApuri koostuu kahdesta osasta: prosessointiohjelmasta, jolla kiinteistörekisterin ja verottajan tiedot yhdistetään ja muotoillaan tarkistettavaan muotoon, sekä QGis lisäosasta, joka tuo tarkastuksen QGis käyttöliittymään karttapohjalle.

Tämä ohje antaa eväät työkalun käyttöön. Ohje kattaa yksittäisten ohjelman painikkeiden toiminnot, sekä listaa tavallisimpien tarkastustehtävien työvaiheet.
