# Asennus ja lisensointi

1. Lataa asennuspaketti
2. Käynnistä QGis ja valitse `Laajennukset -> Laajennusten hallinta ja asennus`
3. Valitse `Asenna laajennus ZIP-tiedostosta` ja osoita ohjelmapaketti
4. Aktivoi ohjelma `Laajennukset`-valikosta. Työkalun kuvake ilmestyy QGis-työkalupalkkiin.  
Nyt QGis-lisäosa on valmis käyttöön. Ensimmäistä kertaa prosessointia käynnistäessä syötä ohjelmalisenssikoodi prosessointiasetuksiin
