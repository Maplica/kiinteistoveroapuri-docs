# Patchnote — 2026-05-12 #1

## Overview
This patch fixes a column misalignment bug that caused incorrect field headers in the `rakennusosa` (12.10.01.21) record group when processing certain tax authority files. The fix is configurable and exposed in the **Asetukset** dialog so users can enable, disable, or adjust it without touching any code.

---

## Bug Fixes

### Column header shift in rakennusosa data (12.10.01.21)
**Cause:** Some versions of the tax authority export file contain an extra column in the rakennusosa record group (record type `12.10.01.21`). Because the data processing pipeline assigns column headers positionally, the extra column caused all subsequent headers to shift by one position — meaning each field was labelled with the name of the field to its left.

**Fix:** After slicing the raw group data, the pipeline now checks the actual column count against the expected count. If exactly one extra column is found and the fix is enabled, the column at the configured index is dropped before headers are assigned.

The logic in `3.0_Tiedonkasittely_to_exe.py` now:
1. Slices with one extra column of headroom when the fix is enabled.
2. Compares actual column count against expected:
   - **Count = expected + 1** → drops the column at the configured index and logs `[Asetukset] Rakennusosa: poistettu ylimääräinen sarake indeksistä N`.
   - **Count = expected** → data is already correct, no action taken.
   - **Any other count** → logs a warning and skips the drop to avoid corrupting data.

**Default behaviour:** fix enabled, extra column at index `6` (zero-based, Excel column G).

## New Features

### Configurable rakennusosa column fix in Asetukset
The rakennusosa column fix is now fully configurable from the **Asetukset** dialog (main window → gear icon) under the new category **"Verottajan tiedostot"**.

Two settings are available:

| Setting | Description | Default |
|---|---|---|
| Poista ylimääräinen sarake rakennusosatiedoista | Enable/disable the column drop entirely | Enabled |
| Sarakkeen indeksi | Zero-based index of the extra column to drop | `6` (Excel column G) |

The spinbox is automatically disabled when the checkbox is unchecked.

Settings are stored in `QSettings` under `Veroselvitysavustaja/Settings/VeroTiedostot/` and are read each time processing is started — no restart required.

#### Where to find it
1. Open the main Veroselvitysavustaja window.
2. Click the **Asetukset** (gear) button or go to **Asetukset** in the menu bar.
3. Select **"Verottajan tiedostot"** in the left-hand category list.

---

## Configuration reference

The two values are written into the processing config JSON as:

```json
{
  "rakennusosa_fix_enabled": true,
  "rakennusosa_extra_col_index": 6
}
```

These can also be set manually in the JSON config if running the exe directly without the plugin.
